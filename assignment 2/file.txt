2. What is DOCTYPE in Html ?
Answer: The `DOCTYPE` in HTML is a special instruction that specifies the version and type of the HTML document.

3. What is Void Element and Example for Void Elements ?
Answer: A void element in HTML is an element that does not have a closing tag and cannot contain any content or other elements. 
eg: <br>, <img >, <hr>

6. What are meta tag and why is it used ?
Answer:Meta tags in HTML are elements used to provide metadata about a web page, including character set, viewport settings, description, keywords, and author information.
